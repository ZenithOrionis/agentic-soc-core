Production API Access

Protected API Links

Production APIs require the X-SOC-API-Key header. Browser clicks to API ports may show missing or invalid SOC API key; that is expected protection.

Use PowerShell with the key from .env.production:

$apiKey = (Select-String -Path .env.production -Pattern "^SOC_API_KEY=").ToString().Split("=")[1]
Invoke-RestMethod -Uri "http://127.0.0.1:8002/incidents" -Headers @{ "X-SOC-API-Key" = $apiKey }
